## AI4S Toolkit: Machine Learning for Materials Science

This repository serves as a personal knowledge base and implementation toolkit for my transition into **AI for Science (AI4S)**.

## 📚 Learning Milestones
- **Machine Learning Foundations:** Systematically completed core modules of [Machine Learning by Andrew Ng](https://www.coursera.org).
- **Data Science Workflow:** Proficient in data cleaning, exploratory data analysis (EDA), and predictive modeling using the Python scientific stack.
- **Computational Physics to ML:** Bridging domain-specific knowledge from [Nature Nanotechnology](https://www.nature.com) studies into data-driven frameworks.

## 🧪 Featured Demos
- **`battery_cycle_regression.ipynb`:** 
  A regression model implementation to predict capacity fading using experimental datasets from my previous studies.
- **`materials_feature_scaling.py`:** 
  Scripts for normalizing multi-scale material properties for gradient-based optimization.

## 🛠 Tech Stack
- **ML Frameworks:** [Scikit-learn](https://scikit-learn.org)
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
